# Contributing to grok-cursor-bridge

Thank you for your interest in improving this tool!

## How to Contribute

### Reporting Issues
- Use the GitHub Issues tab.
- Include:
  - Your operating system (macOS, Windows, Linux)
  - Grok TUI version (if known)
  - Exact command you ran and the output
  - Whether you're using the plugin or manual installation

### Suggesting Improvements
Ideas for better error messages, detection logic, documentation, or examples are very welcome.

### Code Contributions
1. Fork the repository
2. Make your changes
3. Test them thoroughly (especially on your platform)
4. Open a Pull Request with a clear description

### Areas Where Help is Especially Appreciated
- Windows-specific improvements and testing
- Better error messages and edge case handling
- More realistic examples from different tech stacks
- Documentation improvements

## Code of Conduct

Be kind and respectful. This project exists to make people's lives easier when switching between Cursor and Grok.

## Questions?

Feel free to open an issue or reach out on X.
