# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial public release of `grok-cursor-bridge`
- Full plugin support (`plugin.json` + `skills/` layout)
- `status`, `import`, `refresh`, `load`, `list-imported`, and `remove` commands
- Support for `.cursor/agents/`, `.cursor/skills/`, and `.cursorrules`
- Cross-platform installation guidance (macOS, Linux, Windows)
- Comprehensive documentation and examples
- Explicit attribution requirement in the license
- Security & Trust section in README
- `CONTRIBUTING.md`, issue & PR templates
- Improved error handling and user experience guidance in the skill

### Changed
- Restructured repository to follow proper Grok plugin conventions
- Major improvements to README clarity and beginner-friendliness
